# file and contents
### 目录及文件结构
1. 目录采用的数据结构 -- 树结构多级目录
2. 文件存储的逻辑结构 -- 记录文件存储
3. 文件存储物理结构 -- 连续文件存储
4. 文件存储的组织信息 -- 每个文件的记录存储文件体和文件控制块：每个文件控制块包含文件名，上次操作时间，物理地址，存取权限，文件性质，编码方式，文件说明。
5. 运行系统时的总体控制模式
   用户操作->传递给磁盘控制
   命令解释->目录检索->存取控制验证->逻辑文件系统->物理文件系统->操作磁盘文件->反馈给用户
### 提取功能
#### cd 
作用：该改变目录，切换到指定目录

cd[参数][目录名]

cd + 目录名 进入下级目录
cd + . 返回上街目录

#### dir
作用：显示当前目录信息及包含的所有节点

#### mkdir
作用：创建文件夹

#### rmdir
作用：删除文件夹（只能为空）

#### find
作用：找到特定的文件或文件架
